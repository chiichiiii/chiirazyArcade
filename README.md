Java Swing으로 만든 크레이지아케이드 스타일 게임

## 🎮 게임 소개

Chiirazy Arcade는 Java로 제작한 2D 폭탄 액션 게임입니다.  
플레이어는 폭탄을 설치하여 블록을 부수고 상대를 공격할 수 있습니다.

## ✨ 주요 기능

- 2인 로컬 플레이
- AI 봇 플레이
- 꼬물이 모드
- 폭탄 설치 및 연쇄 폭발
- 파워업 아이템 시스템
- 맵 블록 파괴 시스템

## 🕹️ 조작 방법

### Player 1
W A S D → 이동
SPACE → 폭탄 설치


### Player 2
방향키 → 이동
ENTER → 폭탄 설치


## ⚙️ 실행 방법

1. 프로젝트를 클론합니다
git clone https://github.com/yourid/chiirazyArcade.git

2. IntelliJ에서 프로젝트 열기

3. `L30Game.java` 실행

## 🧱 사용 기술

- Java
- Swing
- 2D Tile Map
- BFS AI Pathfinding
