# Arcade Bomber Web

## 실행
- `web-bomber/index.html` 더블클릭으로 바로 실행 가능
- 또는 VSCode Live Server/간단한 정적 서버로 실행

## 조작
- P1: `WASD` 이동, `SPACE` 폭탄
- P2: 방향키 이동, `ENTER` 폭탄
- 공통: `P` 일시정지, `R` 재시작, `1/2` 모드 선택(시작 화면), `ESC` 시작 화면 복귀
